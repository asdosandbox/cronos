# Cronos

Reloj mundial en tiempo real. Una sola página, sin instalación, sin backend.

---

Detecta tu ciudad y zona horaria automáticamente. Muestra la temperatura actual donde estás. Permite comparar la hora de cualquier ciudad del mundo con la tuya, en pantalla completa o en la vista principal.

---

**Hora local · GMT · Temperatura · 165+ ciudades · Fullscreen · Modo oscuro · Cuentas regresivas**
